\section{Example: Binary Search Tree}
\label{sec:bst}
\label{sec:example}

We substantiate the usefulness of our development by applying it to a full BST implementation.
Here, we consider a sequential implementation that uses techniques commonly found in fine-grained concurrent versions, like \cite{DBLP:conf/wdag/FeldmanE0RS18,DBLP:conf/europar/CrainGR13,DBLP:conf/ppopp/BronsonCCO10,DBLP:conf/icdcn/RamachandranM15,DBLP:conf/ppopp/DrachslerVY14}, to name a few.
As a result, our implementation features updates that have an unbounded footprint, i.e., may affect entire subtrees in a way that makes framing inapplicable.
We show how these footprints can be bounded using contextualization.
