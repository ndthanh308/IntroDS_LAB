\section{Chernoff bounds}\label{app:chernoff}
 Let $X_1,X_2,\ldots,X_K$ be independent random variables, where $X_i=1$ with probability $p_i$ and $X_i=0$ with probability $(1-p_i)$. Let $X=\sum_{i=1}^KX_i$ so $E[X]=\sum_{i=1}^Kp_i$. We apply the Chernoff bounds in the following form: For any $0<\delta<1$,
%
\begin{align*}
\Pr[X\geq(1+\delta)E[X]]&\leq e^{-\frac{\delta^2}{3}E[X]}\\
\Pr[X\leq(1-\delta)E[X]]&\leq e^{-\frac{\delta^2}{2}E[X]}
\end{align*}
