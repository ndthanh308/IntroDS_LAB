Given GD with momentum
\begin{align}
    \vm_k &= \beta \vm_{k-1} + (1-\beta)\nabla \Loss(\vx_{k-1})\\
    \vx_k &= \vx_{k-1} - \eta \vm_k\\
\end{align}
Equivalently

\[
\vx_k + \beta \vx_{k-2} - (1+\beta) \vx_{k-1} = -\eta(1-\beta)\nabla\Loss(\vx_{k-1})
\]

Nesterov:
\[
\vx_k + \beta \vx_{k-2} - (1+\beta) \vx_{k-1} = -\eta(1-\beta)\nabla\Loss(\vx_{k-1}) - \eta\beta(1-\beta)(\nabla\Loss(\vx_{k-1})-\nabla\Loss(\vx_{k-2}))
\]
with $\beta=\frac{1-\sqrt{\mu \eta(1-\beta)}}{1+\sqrt{\mu \eta(1-\beta)}}$ ($\mu$-strongly convex) or $\beta=\frac{k-1}{k+2}$ for convex.

Limiting ODE:
\begin{enumerate}
\item Oy: $\beta$ fixed, $\eta\to 0$
\item Al: $\beta\to 1$, $\eta\to 0$, $\frac{1-\beta}{\eta}\to \hatbeta$
\end{enumerate}

Assume:
$\Loss$ has bounded Hessian.

For limit 1 above.
