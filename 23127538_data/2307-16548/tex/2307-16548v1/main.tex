\begin{abstract}
\noindent 
\ifx \undefined \DOCUMENTATION
This article
\else 
This document
\fi 
 presents adequate formal terminology for the mathematical specification of a simplified non-calibrated agent-based demographic model of the UK. Implementation is provided via the Julia package MiniDemographicABM.jl \cite{MiniDemographicABMjlMisc}. 
 Individuals of an initial population are subject to ageing, deaths, births, divorces and marriages. The main purpose of the model is to explore and exploit capabilities of the state-of-the-art Agents.jl Julia package \cite{Datseris2022}. 
Additionally, the model can serve as a base model to be adjusted to realistic large-scale socio-economics, pandemics or social interactions-based studies mainly within a demographic context. A specific simulation is progressed with a user-defined simulation fixed step size on a hourly, daily, weekly, monthly basis or even an arbitrary user-defined clock rate.  
\end{abstract}
